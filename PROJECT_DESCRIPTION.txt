# Descrição Geral do Projeto My Task Vision

Este projeto é um sistema de gerenciamento de tarefas e fluxos de trabalho desenvolvido com React, TypeScript e Supabase. O sistema permite aos usuários criar, gerenciar e acompanhar tarefas, departamentos, processos e fluxos de trabalho em uma interface intuitiva.

## Estrutura Principal:

1. **src/** - Código-fonte principal da aplicação
   - Contém todos os componentes, páginas, hooks e integrações
   
2. **public/** - Arquivos estáticos públicos
   - Imagens, ícones e outros recursos acessíveis publicamente

3. **supabase/** - Configurações e migrações do banco de dados Supabase
   - Definições de esquema, funções e políticas de segurança

## Tecnologias Principais:
- React com TypeScript
- Supabase (Backend-as-a-Service)
- Tailwind CSS para estilização
- shadcn/ui para componentes de interface
- React Query para gerenciamento de estado assíncrono
- React Router para navegação

## Funcionalidades Principais:
- Autenticação de usuários
- Gerenciamento de tarefas (CRUD)
- Organização por departamentos
- Controle de processos e fluxos de trabalho
- Sistema de permissões baseado em papéis
- Visualizações em Kanban e gráficos
- Relatórios diários
