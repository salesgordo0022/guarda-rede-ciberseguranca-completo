# Descrição do Diretório public/

Este diretório contém arquivos estáticos públicos que são servidos diretamente pelo servidor web e podem ser acessados publicamente.

## Arquivos:

1. **favicon.ico**:
   - Ícone favicon exibido na aba do navegador
   - Representa a identidade visual da aplicação

2. **placeholder.svg**:
   - Imagem SVG usada como placeholder quando imagens reais não estão disponíveis
   - Pode ser usada para evitar quebras de layout

3. **robots.txt**:
   - Arquivo de instruções para robôs de busca (web crawlers)
   - Define quais partes do site podem ou não ser indexadas

## Propósito:
- Armazenar arquivos estáticos acessíveis publicamente
- Hospedar recursos como ícones, imagens e documentos
- Servir arquivos sem passar pelo processo de build do JavaScript
- Manter recursos estáticos que não precisam de processamento

## Características:
- Todos os arquivos neste diretório são acessíveis publicamente via URL
- Não são processados pelo bundler (como Vite ou Webpack)
- Mantém a estrutura de diretórios original quando servidos
- Ideal para assets que não mudam frequentemente
