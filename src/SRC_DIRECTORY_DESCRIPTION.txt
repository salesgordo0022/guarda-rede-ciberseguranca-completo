# Descrição do Diretório src/

Este é o diretório principal que contém todo o código-fonte da aplicação My Task Vision.

## Estrutura do Diretório:

1. **components/** - Componentes reutilizáveis da interface do usuário
   - Componentes personalizados da aplicação
   - Componentes UI (shadcn/ui) na subpasta ui/

2. **hooks/** - Hooks React personalizados
   - Hooks para autenticação, permissões, configurações do sistema e gerenciamento de tarefas

3. **integrations/** - Integrações com serviços externos
   - Integração principal com Supabase (client, mock e tipos)

4. **lib/** - Funções utilitárias e bibliotecas auxiliares
   - Funções utilitárias compartilhadas

5. **pages/** - Páginas da aplicação (rotas)
   - Cada arquivo representa uma página/route específica da aplicação

6. **types/** - Definições de tipos TypeScript
   - Interfaces e tipos personalizados usados em toda a aplicação

7. **utils/** - Funções utilitárias específicas
   - Utilitários relacionados a tarefas e outras funcionalidades específicas

## Arquivos Principais:
- App.tsx - Componente raiz da aplicação
- main.tsx - Ponto de entrada da aplicação
- index.css - Estilos globais
