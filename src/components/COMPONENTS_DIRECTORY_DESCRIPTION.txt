# Descrição do Diretório components/

Este diretório contém todos os componentes reutilizáveis da interface do usuário da aplicação My Task Vision.

## Tipos de Componentes:

1. **Componentes de UI Personalizados**:
   - ActionButtons.tsx - Botões de ação para tarefas
   - ActivitiesTable.tsx - Tabela para exibir atividades
   - AppSidebar.tsx - Barra lateral principal da aplicação
   - CreateTaskDialog.tsx - Diálogo para criação de tarefas
   - DailyReportDialog.tsx - Diálogo para relatórios diários
   - DashboardCard.tsx - Cartões do dashboard
   - DepartmentPanel.tsx - Painel de gerenciamento de departamentos
   - DepartmentTasksPanel.tsx - Painel de tarefas por departamento
   - ErrorBoundary.tsx - Componente para tratamento de erros
   - KanbanBoardDialog.tsx - Visualização Kanban das tarefas
   - NavLink.tsx - Componente de navegação
   - PermissionGuard.tsx - Guarda de permissões
   - TaskChart.tsx - Gráficos de tarefas
   - TaskHistoryTable.tsx - Tabela de histórico de tarefas
   - TaskTable.tsx - Tabela principal de tarefas
   - TeamTabContent.tsx - Conteúdo da aba de equipe nas configurações

2. **Componentes UI (shadcn/ui)**:
   - Pasta ui/ contém componentes prontos como botões, cartões, tabelas, diálogos, etc.
   - Esses componentes são baseados na biblioteca shadcn/ui e estilizados com Tailwind CSS

## Propósito:
- Promover a reutilização de código
- Manter consistência na interface do usuário
- Facilitar a manutenção e atualização da UI
- Separar a lógica de apresentação da lógica de negócios
