# Descrição do Diretório pages/

Este diretório contém todas as páginas da aplicação, onde cada arquivo representa uma rota específica no sistema de gerenciamento de tarefas.

## Páginas Disponíveis:

1. **Activities.tsx** - Página principal de atividades por departamento
2. **Auth.tsx** - Página de autenticação/login
3. **Departments.tsx** - Gerenciamento de departamentos
4. **GenerateActivity.tsx** - Geração de novas atividades
5. **GenerateProcess.tsx** - Geração de novos processos
6. **GenerateWorkflow.tsx** - Geração de novos fluxos de trabalho
7. **Index.tsx** - Página inicial/dashboard
8. **NotFound.tsx** - Página de erro 404
9. **ProcessControl.tsx** - Controle de processos
10. **ProcessGenerator.tsx** - Gerador avançado de processos
11. **Processes.tsx** - Listagem e gerenciamento de processos
12. **Settings.tsx** - Configurações do sistema
13. **TaskGenerator.tsx** - Gerador avançado de tarefas
14. **Tasks.tsx** - Gerenciamento principal de tarefas
15. **Team.tsx** - Gerenciamento de equipe e usuários
16. **Users.tsx** - Gerenciamento de usuários
17. **Workflows.tsx** - Listagem e gerenciamento de fluxos de trabalho

## Propósito:
- Cada arquivo representa uma rota específica na aplicação
- Contém a lógica de apresentação e interação do usuário
- Conecta componentes reutilizáveis com a lógica de negócios
- Define o layout e estrutura de cada seção da aplicação

## Características:
- Utilizam componentes do diretório components/
- Consomem hooks do diretório hooks/
- Implementam as principais funcionalidades do sistema
- São acessadas através da navegação da aplicação
